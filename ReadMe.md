# Network Intrusion Detection Agent 🚀

AI-driven Network Intrusion Detection System with interactive visualization and prediction interface.

This repository contains:
- Supervised ML models (Random Forest & XGBoost) for intrusion detection
- Multi-stage classification pipelines
- A modern Streamlit dashboard for uploading network logs and visualizing predictions
- Utilities for training, evaluation, and feature preprocessing

---

## 🔍 Features

- **Stage 1:** Binary classification (Benign vs Attack)
- **Stage 2:** Fine-grained attack type classification (multi-class)
- **Interactive Dashboard:** Upload CSV datasets and run predictions
- **Evaluation Tools:** Confusion matrix, performance metrics, feature importance
- **Clean preprocessing:** Handling missing and infinite values
